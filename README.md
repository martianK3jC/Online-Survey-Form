# Online Survey Form

**Live demo:** [https://martiank3jc.github.io/Online-Survey-Form/](https://martiank3jc.github.io/Online-Survey-Form/)

## Overview

Online Survey Form is a simple front-end web project built with HTML, CSS, and JavaScript. It allows users to submit feedback through a survey form, preview their input live, and view submitted responses in a table.

The project is designed to demonstrate basic JavaScript DOM manipulation, event handling, form validation, dynamic content updates, and CSS class manipulation.

## Features

- Online survey form with user input fields
- Live preview that updates while the user types or selects a rating
- Form validation before submission
- Error message display for invalid or missing input
- Invalid submissions blocked until errors are corrected; brief busy/disabled state on the submit button while processing
- Summary section showing total responses and average rating
- Responses table that displays submitted survey entries
- DOM updates using JavaScript
- CSS class manipulation for visual feedback
- Attribute manipulation for dynamic form behavior

## Project Requirements

The project should include:

- A page header
- A feedback form section
- A live preview section
- A summary section
- A responses table section
- JavaScript functionality for live updates and form validation
